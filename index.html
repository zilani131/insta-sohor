2.Differentiate between local storage and session storage

Local storage Session storage
We can delete local storage data by javascript code or manually Session storage data is deleted when we close the window or tab
We can access the data from any window. We can only access the data only the same window
For saving data in local storage we write
localStorage.setItem('key', 'value');		 For saving data in session storage we write
rsessionStorage.setItem('key', 'value');
For getting item from local storage we write;
let data = localStorage.getItem('key');		 For getting item from local storage we write;
let data = sessionStorage.getItem('key');
For removing an item we write;
localStorage.removeItem(key); 		For removing an item we write;
sessionStorage.removeItem(key);
